About Me 💖

Hi, I’m Alex! I’m a data scientist based in Ottawa, Ontario. I’m an organizer for R-Ladies Ottawa and am an avid knitter, sewist, and fibre artist 🧶

Find my (textile) work on my ravelry and my (technical) work on my GitHub and personal site

My submission ⚒️

Ravelry is a website for fibre artists to track their projects, browse, and sell knitting or crochet patterns. Ravelry has a “hot right now” section which features the patterns with the most visits in the last 24 hours.

In this dashboard I fetch the Ravelry “hot right now” top 20 and present pattern details in a convenient and easy to read table so fibre artists can quickly decide if they’re interested in a pattern. I remove any patterns related to knit/crochet-alongs as I personally find them really annoying and think the page traffic for such patterns is somewhat artificially inflated anyways.

  • Technical details 👩‍💻
  • Further work 🚦

I used the ravelRy R package by Kaylin Pavlik to easily access the Ravelry api. I wrote some R code to fetch the 20 top “hot right now” patterns, some additional details about the patterns and their designers, and used the gt package to display the results in a beautiful table hosted in a Quarto dashboard with GitHub pages. Using my Github action (which runs automatically each day) I store the most recent data so I can check the history and see if a pattern has ranked up or down. I also included a 7-day nanoplot to see a pattern’s rank history. Finally, I used CSS and JS to add a lightbox to view a bigger version of each photo in the table.

I am not overly familiar with (nor do I enjoy working with) CSS or JS so I borrowed some style code (which I then modified to suit my needs) from my friend, colleague, R-Ladies co-organizer, and last year’s table contest winner, Melissa Van Bussel.

I’ve played with the Ravelry API before! Check out my R-Ladies Ottawa 2025 data hackathon submission where I examined my own knitting data in another quarto dashboard.

I’m inspired by the intersection of math, stats, data science and textile arts. Here are some projects/artists that I find inspiring:

  • clay_muchi on Instagram: Topographical crochet.
  • Daina Taimina: Inventor of the crocheted hyperbolic model.
  • Typeknitting: An exploration of the typographical potential of handknitting.
  • Aperiodic Einstein Monotile: A knitting pattern for the new (ish) monotile.
  • Copola Views Quilt: Quilt pattern inspired by the view from the ISS
